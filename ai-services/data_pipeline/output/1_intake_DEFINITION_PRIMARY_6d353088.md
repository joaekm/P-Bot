# Definition: Grafisk formgivare
En roll inom Kompetensområde 6.

## Rollbeskrivning
Tar fram den grafiska formen för ett system eller en webbplats. Arbetar med interaktionsdesigner och utvecklare för att skapa ett tydligt och lättanvänt gränssnitt som överensstämmer med organisationens grafiska profil.

## Efterfrågad kompetens
Kunskap och erfarenhet av att skapa gränssnitt med hög användbarhet som stämmer överens med uppdragsgivarens grafiska profil.

## Exempel på uppdrag
- Ta fram den grafiska formen för ett system i linje med myndighetens grafiska profil.
