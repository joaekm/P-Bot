# Regel: Förbud mot Alternativa Anbud och Reservationer
Enligt allmänna upphandlingsprinciper och villkor i avropsförfrågan är det inte tillåtet att lämna sidoanbud, alternativa anbud, eller att reservera sig mot ställda villkor.

**Konsekvens:** Anbud som innehåller förbehåll eller reservationer kan komma att förkastas.
