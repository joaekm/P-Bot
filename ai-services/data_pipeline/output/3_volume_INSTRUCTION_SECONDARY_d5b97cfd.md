# Instruktion: Faktureringskrav
Fakturering ska ske månadsvis i efterskott. Fakturor SKA vara elektroniska (e-fakturor) och skickas via PEPPOL-nätverket. PDF-fakturor accepteras ej. 

**Obligatoriska uppgifter på fakturan:**
- Specifikation av utfört arbete
- Referens/kostnadsställe: 9015
- Kontraktets diarienummer: 2023090010
