# Instruktion: Specificera Krav på Konsult
DEFINIERA tydliga och verifierbara krav på resurskonsulten.

**Checklista:**
- **Uppdragsbeskrivning:** Vad ska konsulten göra?
- **Roll:** Vilken roll/kompetens efterfrågas?
- **Kompetensnivå:** Ange nivå 1-5.
- **Erfarenhetskrav:** Specificera krav på erfarenhet, kompetens och referenser.
- **Bevisning:** Ange hur krav ska styrkas, t.ex. via CV eller referensuppdrag.
