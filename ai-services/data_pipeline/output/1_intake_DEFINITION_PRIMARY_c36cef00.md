# Definition: Kompetensområde 7 – Användarsupport/Teknikstöd
Detta kompetensområde omfattar uppdrag där support ges antingen på distans (telefon, fjärranslutning) eller på plats hos användaren. Området innehåller särskilda regler för prissättning och avropsform.
