# Instruktion: Aktivering av Option
**PROCESS:**
1.  Om den avropande myndigheten vill använda en option SKA leverantören meddelas skriftligen.
2.  Meddelandet måste ske innan den tidsfrist som angetts i kontraktet för optionen löper ut.
