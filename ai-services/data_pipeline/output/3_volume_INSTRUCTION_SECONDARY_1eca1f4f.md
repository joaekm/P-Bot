# Instruktion: Krav på e-faktura och referenser
Fakturering ska ske med elektronisk faktura (e-faktura), t.ex. via PEPPOL. Fakturor måste innehålla specifika referenser, såsom kostnadsställe och diarienummer, för att kunna hanteras.
