# Instruktion: Ange Ersättningsmodell
Kontraktet måste specificera vilken ersättning leverantören har rätt till. Ange ersättningsform (fast pris, löpande räkning etc.) samt hur och när betalning ska ske. Vid FKU hänvisas till avropsvägledningens avsnitt 4.2.3 för vägledning om ersättningsformer.
