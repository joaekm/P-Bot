# Instruktion: Elektronisk Anbudsgivning
Anbud ska lämnas elektroniskt via upphandlingssystemet. I detta exempel används Mercell TendSign.

**Process:**
1.  Registrera ett konto på www.tendsign.com.
2.  Ladda ner samtliga upphandlingsdokument.
3.  Ställ eventuella frågor via systemets Q&A-funktion före sista frågedag.
4.  Ladda upp och skicka in komplett anbud elektroniskt innan anbudstidens utgång.
