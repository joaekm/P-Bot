# Regel: Förbud mot alternativa anbud
I anbud inom offentlig upphandling får inga så kallade sidoanbud eller alternativa anbud förekomma, om det inte uttryckligen tillåts i upphandlingsdokumentet. Om anbudsgivaren lämnar förbehåll eller reserverar sig mot ställda villkor kan anbudet komma att förkastas.
