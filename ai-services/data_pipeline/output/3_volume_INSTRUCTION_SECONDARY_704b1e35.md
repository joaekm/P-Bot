# Instruktion: Krav för Fakturering
Fakturor ska skickas elektroniskt, primärt via PEPPOL-nätverket. Varje faktura måste innehålla specifikation av utfört arbete, korrekt referens/kostnadsställe och kontraktets diarienummer.
