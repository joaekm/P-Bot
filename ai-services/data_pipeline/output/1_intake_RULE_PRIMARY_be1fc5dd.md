# Regel: Språkkrav för Anbud
**KRAV:** Anbudet SKA vara skrivet på svenska.
**UNDANTAG:** Bevis, vedertagna begrepp, förkortningar och tekniska beskrivningar FÅR vara på engelska.
**ANSVAR:** Om avropande myndighet begär översättning SKA Ramavtalsleverantören bekosta detta.
