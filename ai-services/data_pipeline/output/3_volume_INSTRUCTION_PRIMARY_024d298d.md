# INSTRUCTION: Ansvar för kommunikation vid prisjustering
Ansvaret för att meddela prisjusteringar skiljer sig åt:
1.  **Ramavtalsnivå:** Adda Inköpscentral meddelar leverantören.
2.  **Kontraktsnivå (efter FKU):** Leverantören meddelar den avropande upphandlande myndigheten.
