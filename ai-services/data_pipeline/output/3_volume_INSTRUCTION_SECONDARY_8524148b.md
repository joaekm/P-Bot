# Instruktion: Krav på Elektronisk Fakturering
Beställaren tar endast emot elektroniska fakturor, primärt via nätverket PEPPOL. PDF-fakturor accepteras ej. Varje faktura måste innehålla ett specifikt referensnummer/kostnadsställe (i detta exempel '8431') och kontraktets diarienummer.
