# Instruktion: Krav på Elektronisk Fakturering (PEPPOL)
Beställaren (Inera) accepterar endast elektroniska fakturor (e-fakturor). PDF-fakturor via e-post godkänns ej. Fakturor ska sändas via nätverket PEPPOL eller via VAN-tjänst. Fakturan måste innehålla specifik referensinformation som kostnadsställe och diarienummer.
