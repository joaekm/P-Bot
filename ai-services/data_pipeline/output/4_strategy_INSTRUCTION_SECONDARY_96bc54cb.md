# Instruktion: Hantering av frågor och anbudsgivning
Processen för anbudsgivning sker elektroniskt via ett upphandlingssystem (t.ex. Mercell TendSign). Anbudsgivare ansvarar för att hålla sig uppdaterade i systemet.

**Process:**
1.  **Registrering:** Anbudsgivare måste registrera sig i systemet för att få fullständig information.
2.  **Frågor & Svar:** Frågor om upphandlingsdokumentet ska ställas via systemet innan en angiven tidsfrist. Svar publiceras till alla.
3.  **Inlämning:** Anbud ska lämnas in elektroniskt via systemet innan anbudstidens utgång.
