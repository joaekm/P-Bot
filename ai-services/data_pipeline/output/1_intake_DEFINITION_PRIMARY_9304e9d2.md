# Definition: Kompetensområde 4 – Arkitektur
Arkitektrollerna beskrivs primärt ur ett IT-perspektiv, men roller som Enterprisearkitekt och Verksamhetsarkitekt kan arbeta med ett bredare verksamhetsfokus. Tjänsterna kan avropas för både linje- och projektarbete. Beskrivningarna baseras på IASA Swedens beskrivningar från 2020.
