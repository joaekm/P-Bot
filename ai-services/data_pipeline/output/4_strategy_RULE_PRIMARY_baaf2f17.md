# Regel: Svarstid vid avrop
Leverantörer SKA svara på avropsförfrågningar inom fastställda tidsfrister:

*   **Dynamisk Rangordning:** Inom **2 arbetsdagar** (om inte längre tid anges).
*   **Förnyad Konkurrensutsättning:** Inom **10 arbetsdagar** (om inte längre tid anges).

Avropande myndighet BÖR anpassa svarstiden vid komplexa förfrågningar.
