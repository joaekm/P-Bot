# Definition: Uppdragstyper
Avrop kan specificeras med en av två uppdragstyper som definierar vem som leder uppdraget:
*   **Resursuppdrag:** Beställaren leder uppdraget.
*   **Projektuppdrag:** Leverantören leder uppdraget och ansvarar för att leverera ett bestämt resultat.
