# Regel: Sekretess
Leverantören förbinder sig att följa bestämmelser om sekretess enligt Offentlighets- och sekretesslagen (2009:400) och annan tillämplig lagstiftning. Inera har rätt att kräva att enskilda konsulter signerar ett separat sekretessavtal för att få påbörja uppdraget.
