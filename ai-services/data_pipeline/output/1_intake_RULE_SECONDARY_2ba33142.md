# Regel: Användning av Underleverantörer
Användning av underleverantörer för att utföra delar av ett uppdrag är tillåtet, men kräver skriftligt godkännande från beställaren (i detta fall Inera). Ramavtalsleverantören behåller fullt ansvar för underleverantörens arbete, som om det vore deras eget.
