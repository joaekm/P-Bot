# Regel: Användning av Underleverantörer
**KRAV:** Användning av underleverantörer för att utföra delar av uppdraget kräver skriftligt godkännande från beställaren (Inera) i förväg.

**ANSVAR:** Ramavtalsleverantören är alltid fullt ansvarig för underleverantörens arbete, som om det vore deras eget.
