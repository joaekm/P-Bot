# Instruktion: Obligatoriska fakturareferenser
För att möjliggöra automatiserad avstämning MÅSTE en faktura innehålla följande referenser:
- Kontraktsnummer/kontraktsreferens
- Namn/projektnummer på ramavtalet
- Avrops-/beställnings-/inköpsordernummer
- Kostnadsställe eller beställarens referens
