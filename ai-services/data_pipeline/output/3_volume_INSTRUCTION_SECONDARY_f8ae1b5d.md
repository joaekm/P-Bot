# Instruktion: Fakturering
Fakturor SKA skickas elektroniskt. Pappers- eller PDF-fakturor accepteras ej. Inera kan ta emot e-fakturor via PEPPOL-nätverket. 

**Obligatorisk information på fakturan:**
- Specifikation av utfört arbete
- Korrekt referens/kostnadsställe
- Kontraktets diarienummer
