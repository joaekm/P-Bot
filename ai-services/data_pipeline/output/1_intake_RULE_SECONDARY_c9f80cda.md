# Regel: Sekretess enligt Offentlighets- och sekretesslagen (OSL)
Leverantören förbinder sig att följa bestämmelser om sekretess i enlighet med `Offentlighets- och sekretesslagen (2009:400)`. Den upphandlande myndigheten (Inera) gör den slutgiltiga prövningen av vilka uppgifter i ett anbud som kan sekretessbeläggas efter att upphandlingen avslutats.
