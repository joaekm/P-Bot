# Instruktion: Krav på Elektronisk Fakturering
Kunden (Inera AB) tar endast emot elektroniska fakturor, primärt via nätverket PEPPOL. PDF-fakturor accepteras ej. Fakturan måste innehålla specifika referenser som kostnadsställe och diarienummer.
