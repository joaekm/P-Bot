# Definition: Prismodeller och Nivåer
Ramavtalet har fasta timpriser för definierade exempelroller på kompetensnivå 1-4. För övriga uppdrag, som de som prissätts via FKU, finns möjlighet att använda andra prismodeller för att uppnå förutsägbara kostnader.
