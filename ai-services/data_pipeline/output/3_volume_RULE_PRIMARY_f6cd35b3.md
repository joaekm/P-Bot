# Regel: Ersättning för Obekväm Arbetstid
Normal arbetstid definieras som helgfria vardagar kl. 07:00-19:00. För arbete utanför denna tid ska kontrakterat timpris multipliceras enligt:
- **OB Tillägg 1 (vardagar 06:00-07:00 & 19:00-21:00):** Timpris x 1.5
- **OB Tillägg 2 (övrig tid, inkl. helger):** Timpris x 2.0

Detta gäller ej för uppdrag som inkluderar beredskap, vilket ska regleras separat i kontraktet.
