# Definition: Kompetensområde 3 – Systemutveckling/Systemförvaltning och infrastruktur
Detta kompetensområde omfattar uppdrag inom systemutveckling, systemintegration och förvaltning av befintliga system. Det inkluderar även uppdrag inom infrastruktur (lokal, molnbaserad eller hybrid). Inriktningar kan vara BI, Big Data, IoT, AI etc. Uppdragen kräver ofta kunskap om modeller som ITIL, SCRUM och DevOps. Licens- och hårdvarukostnader kan förekomma.
