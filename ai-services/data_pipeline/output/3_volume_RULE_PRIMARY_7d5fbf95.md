# Regel: Ersättning för Obekväm Arbetstid
Om avropande myndighet begär arbete utanför normaltid (07:00-19:00, helgfria vardagar), tillämpas följande multiplikatorer på det kontrakterade timpriset:

| Tidpunkt | Multiplikator |
|---|---|
| Vardag 06:00-07:00 | x 1.5 |
| Vardag 19:00-21:00 | x 1.5 |
| Övrig tid (natt/helg) | x 2.0 |
