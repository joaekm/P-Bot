# Specificera krav på placering i avropsförfrågan

Avropande myndigheter har frihet att utforma sina avrop, inklusive krav på var konsulten ska sitta (på plats, distans eller kombination), så länge det inte strider mot ramavtalets villkor eller LOU. Specificera önskemål i avropsförfrågan.

# Specificera geografiskt område vid avrop

Vid avrop bör den avropande myndigheten specificera vilket geografiskt område avropet gäller för att säkerställa att rätt leverantörer kontaktas och att avtalet följs. Detta kan göras genom att ange delområde (A-G) eller de specifika län som avropet avser i avropsförfrågan.
