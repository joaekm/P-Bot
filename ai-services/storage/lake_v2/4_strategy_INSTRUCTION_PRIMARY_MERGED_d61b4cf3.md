# Val av avropsmetod: Förnyad konkurrensutsättning eller Direktavrop

Vid avrop från ramavtalet IT-konsulttjänster 2021 kan avrop ske genom förnyad konkurrensutsättning (FKU) eller direktavrop (DR). Avropande myndigheter är fria att utforma sina avrop hur de vill så länge dessa inte strider mot ramavtalets villkor eller mot lagen om offentlig upphandling (LOU).
