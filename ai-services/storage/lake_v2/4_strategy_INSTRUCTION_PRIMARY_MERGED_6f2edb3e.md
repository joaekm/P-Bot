# Utvärderingskriterier vid förnyad konkurrensutsättning

Vid förnyad konkurrensutsättning kan följande kriterier användas för att utvärdera anbud och bedöma konsulters lämplighet:

*   **Pris:** Timpris eller fast pris för uppdraget.
*   **Kvalitet:**
    *   Funktion: Beskrivning av förväntat resultat eller användarvänlighet.
    *   Kompetens och erfarenhet hos nyckelpersoner för uppdraget.
    *   Kundnöjdhet: Referenser från tidigare uppdrag.
    *   Leveranstid för tjänster.

Utvärderingsmodellen kan bygga på en mervärdesmodell där prisavdrag erhålls för erbjuden kvalitet. Intervjuer kan användas för att bedöma mervärden som kommunikativ förmåga och förmåga att förmedla kunskaper.
