# Begära in referensuppdrag / Referensuppdrag vid resursavrop

Vid avrop av resurser kan referensuppdrag användas för att styrka konsultens tidigare erfarenhet. För att styrka tidigare erfarenhet kan du begära in beskrivningar av referensuppdrag. I avropsförfrågan ska följande framgå:

*   Beskrivning av referensuppdrag
*   Uppdragsgivare
*   Kontaktperson hos uppdragsgivare
*   Telefonnummer till kontaktperson
*   E-postadress till kontaktperson

För att kontrollera att angivna uppgifter är korrekta kan referenten komma att kontaktas via e-post eller telefon. Referenten ska vara vidtalad.

Vid krav på referenser i en avropsförfrågan, specificera antalet referenser som önskas och vilken typ av uppdrag de ska vara från. Om relevant, ange att referenserna ska vara från liknande uppdrag inom fastighetsbolag.

Exempel på information som bör framgå för varje referens:

*   Beskrivning av referensuppdrag
*   Uppdragsgivare
*   Kontaktperson hos uppdragsgivare
*   Telefonnummer till kontaktperson
*   E-postadress till kontaktperson

Det bör även anges att referenten ska vara vidtalad och att den avropande myndigheten kan komma att kontakta referenten för att kontrollera uppgifterna.

Krav på tidigare erfarenhet ska styrkas av till avropssvaret bilagt CV. Konsulten/konsulterna ska ha erfarenhet av ett liknande uppdrag. Detta styrks genom att ramavtalsleverantören till anbudet bifogar en beskrivning av ett motsvarande uppdrag. Följande uppgifter ska framgå:
* Beskrivning av referensuppdrag
* Uppdragsgivare
* Kontaktperson hos uppdragsgivare
* Telefonnummer till kontaktperson
* E-postadress till kontaktperson
