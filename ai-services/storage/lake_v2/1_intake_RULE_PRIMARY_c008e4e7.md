# Kompetenskrav för Mjukvaru- och Infrastrukturarkitekter

Dokumentet specificerar inte ytterligare tekniska färdigheter och certifieringar utöver de som redan nämns för rollerna som mjukvaruarkitekt och infrastrukturarkitekt. Fördjupad kompetens kan inkludera expertis inom specifika teknologier, ramverk eller plattformar relevanta för uppdraget.
