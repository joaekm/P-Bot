# Val av avropsform vid specifika krav på Cosmic-erfarenhet

Vid specifika krav på Cosmic-erfarenhet rekommenderas att använda **Förnyad Konkurrensutsättning (FKU)**. Detta möjliggör att utvärderingskriterierna kan anpassas för att säkerställa att leverantören har den erfarenhet som krävs.
