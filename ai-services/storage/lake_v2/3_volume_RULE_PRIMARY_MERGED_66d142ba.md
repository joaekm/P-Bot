# Avtalsperiod och förlängning

Information om maximal förlängningsperiod för konsultuppdrag specificeras inte i detta dokument. Avropande myndigheter är fria att utforma sina avrop hur de vill så länge dessa inte strider mot ramavtalets villkor eller mot lagen om offentlig upphandling (LOU).
