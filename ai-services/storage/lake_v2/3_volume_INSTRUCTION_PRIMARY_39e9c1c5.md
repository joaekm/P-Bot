# Utvärdering av anbud utan specificerad budget

Om budgeten inte är specificerad i avropsförfrågan, måste utvärderingen av anbuden fokusera på andra utvärderingskriterier än pris. Vid förnyad konkurrensutsättning kan kriterier som kvalitet, kompetens, erfarenhet, kundnöjdhet och leveranstid användas för att avgöra vilket anbud som är det ekonomiskt mest fördelaktiga. Utvärderingsmodeller som mervärdesmodeller kan också användas, där prisavdrag erhålls för erbjuden kvalitet.
