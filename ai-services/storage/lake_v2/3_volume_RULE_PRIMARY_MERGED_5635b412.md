# Flexibilitet i konsultanvändning

Avropande myndigheter är fria att utforma sina avrop hur de vill så länge dessa inte strider mot ramavtalets villkor eller mot lagen om offentlig upphandling (LOU). Detta innebär att det finns flexibilitet i hur många timmar konsulterna används per vecka, så länge det ryms inom ramavtalet och LOU.
