# Takpris vid avrop

Vid avrop kan ett takpris för uppdraget förhandlas fram. Detta ger en övre gräns för kostnaderna och skyddar mot eventuella överskridanden. Ett takpris kan vara lämpligt om det finns osäkerhet kring uppdragets exakta omfattning.
