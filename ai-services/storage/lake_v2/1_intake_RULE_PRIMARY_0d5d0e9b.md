# Viktiga certifieringar och säkerhetskrav

För vissa roller kan specifika certifieringar eller säkerhetskrav vara viktiga. Detta kan inkludera krav på säkerhetskontroll av den konsult som ska utföra uppdraget. Bevis på kunskap och erfarenhet kan krävas genom referenser, intervjuer eller CV.
