# Fördelning av uppskattade timmar mellan N och [ANONYMIZED_1]-[ANONYMIZED_1]

För att fördela de uppskattade 500 timmarna mellan N och [ANONYMIZED_1]-[ANONYMIZED_1], följ dessa steg:

1.  **Identifiera relevanta roller:** Utgå från rollistan i dokumentet `it-konsulttjanster-2021-v4.0-2022-11-21.xlsx` för att identifiera vilka specifika roller som ingår i kategorierna N och [ANONYMIZED_1]-[ANONYMIZED_1].
2.  **Specificera timmar per roll:** Bestäm hur många timmar som ska allokeras till varje roll inom respektive kategori. Detta kan baseras på projektets behov och komplexitet.
3.  **Summera timmar per kategori:** Summera de allokerade timmarna för alla roller inom N respektive [ANONYMIZED_1]-[ANONYMIZED_1] för att få den totala tidsfördelningen.

Exempel: Om N består av rollerna 'Projektledare' (200 timmar) och 'Systemutvecklare' (100 timmar), och [ANONYMIZED_1]-[ANONYMIZED_1] består av 'Verksamhetsarkitekt' (100 timmar) och 'Lösningsarkitekt' (100 timmar), blir fördelningen:

*   N: 300 timmar
*   [ANONYMIZED_1]-[ANONYMIZED_1]: 200 timmar
