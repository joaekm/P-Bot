# Prissättning kan påverkas av plats

Prissättningen för IT-konsulttjänster kan påverkas beroende på om arbetet utförs på plats eller på distans. Faktorer som kan påverka kostnaden inkluderar resekostnader och andra omkostnader som kan tillkomma vid arbete på plats. Prisjusteringar kan ske enligt ramavtalet.
