# Säkerställa samarbete mellan konsulter

För att säkerställa effektivt samarbete och enhetliga lösningsförslag mellan konsulter:

*   **Definiera tydliga roller och ansvar:** Klargör vem som ansvarar för vad i uppdraget.
*   **Etablera kommunikationskanaler:** Skapa regelbundna möten och använd verktyg för samarbete och informationsdelning.
*   **Fastställ gemensamma mål:** Se till att båda konsulterna förstår och är överens om målen för uppdraget.
*   **Använd projektuppdrag:** Om möjligt, strukturera uppdraget som ett projektuppdrag där leverantören har huvudansvaret för att leverera ett visst resultat, vilket kan främja samarbete.
*   **Gemensam kravspecifikation:** Involvera båda konsulterna i framtagningen av kravspecifikationer för att säkerställa en gemensam förståelse.
*   **Utvärdera samarbetsförmåga:** Inkludera samarbetsförmåga som ett utvärderingskriterium vid val av konsulter.
