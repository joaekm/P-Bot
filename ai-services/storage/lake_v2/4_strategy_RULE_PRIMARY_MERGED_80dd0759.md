# Möjlighet till vite eller sanktioner vid bristande leverans (förnyad konkurrensutsättning)

Det är möjligt att inkludera villkor om vite eller andra sanktioner i kontraktet vid förnyad konkurrensutsättning om konsulten inte lever upp till de ställda kraven. Detta kan vara ett sätt att säkerställa att leverantören tar sitt åtagande på allvar och levererar enligt avtal.
