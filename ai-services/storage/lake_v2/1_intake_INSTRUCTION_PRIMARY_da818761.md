# Val av avropsförfarande påverkar möjligheten till distansarbete

Vid avrop via dynamisk rangordning eller förnyad konkurrensutsättning kan den upphandlande myndigheten specificera kraven för uppdraget. Detta innebär att de kan ställa krav på om konsulten ska sitta på plats, eller om uppdraget kan utföras på distans eller hybrid. 

*   **Dynamisk rangordning:** Används för kortare uppdrag (under 320 timmar). Här specificeras roll och kompetensnivå.
*   **Förnyad konkurrensutsättning:** Används för längre uppdrag (över 320 timmar), kompetensnivå 5, eller om en egen roll formuleras. Här kan den upphandlande myndigheten ange krav och utvärderingskriterier.
