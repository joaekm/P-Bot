# Fastställande av budget och takpris vid avrop

För att fastställa budget eller takpris för ett avrop, använd följande metoder:

1.  **Dynamisk rangordning:** Konsultera dokumentet `efterfragat-resursbehov-bilaga_69e7.md` för information om prissättning och totala priser baserat på rangordning.
2.  **Förnyad konkurrensutsättning:** Utvärdera anbud baserat på de utvärderingskriterier som fastställts i avropsförfrågan (se `avropsvagledning---it-konsultt_35df.md`).
3.  **Generella prisvillkor:** Beakta eventuella generella prisvillkor som anges i ramavtalet, inklusive ersättning för obekväm tid (se `avropsvagledning---it-konsultt_35df.md`).

Dokumentet `efterfragat-resursbehov-bilaga_69e7.md` innehåller även information om indexprissättning som kan påverka den slutliga kostnaden.
