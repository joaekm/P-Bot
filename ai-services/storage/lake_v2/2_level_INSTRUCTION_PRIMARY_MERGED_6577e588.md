# Bedömning av kompetensnivå baserat på uppdragets komplexitet och självständighet

För att bedöma vilken kompetensnivå (1-5) som krävs för ett uppdrag, beakta följande faktorer:

1.  **Komplexitet:** Hur komplexa är arbetsuppgifterna och problemen som konsulten förväntas hantera?
2.  **Självständighet:** I vilken grad förväntas konsulten arbeta självständigt utan arbetsledning?
3.  **Ansvar:** Vilket ansvar har konsulten för att leverera resultat och leda andra?

Jämför dessa faktorer med definitionerna för varje kompetensnivå för att fastställa lämplig nivå.
