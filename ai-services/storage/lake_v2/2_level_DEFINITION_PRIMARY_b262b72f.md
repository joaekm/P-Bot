# Kompetensnivå för testledare

Källmaterialet specificerar inte en specifik kompetensnivå för rollen som testledare. Däremot beskrivs generella kompetensnivåer (1-5) som kan appliceras på olika konsultroller, inklusive testledare, beroende på uppdragets krav och komplexitet. Se beskrivningarna för nivå 1-5 i dokumentet för att bedöma vilken nivå som krävs.
