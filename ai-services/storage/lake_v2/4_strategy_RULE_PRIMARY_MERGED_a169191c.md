# Konsekvenser vid bristande kompetens eller prestation

Om konsulten inte lever upp till kraven på kompetens och erfarenhet, eller om bytet av konsult orsakar förseningar eller ytterligare kostnader för den upphandlande myndigheten, kan det betraktas som ett avtalsbrott. Detta kan leda till sanktioner enligt avtalet, såsom vite eller i allvarliga fall hävning av avtalet.

# Påföljder vid bristande kravuppfyllelse

Dokumentationen ger inte specifik information om påföljder vid avtalsbrott. Generellt kan avtalsbrott leda till krav på rättelse, skadestånd eller i allvarliga fall hävning av avtalet. Se till att inkludera tydliga påföljder i kontraktet vid förnyad konkurrensutsättning.
