# Definition av kompetensområde inom IT-konsulttjänster

Ett kompetensområde inom IT-konsulttjänster definierar ett specifikt område av expertis eller specialisering som en konsult kan erbjuda. Adda ramavtalet för IT-konsulttjänster 2021-2025 omfattar sju kompetensområden.
