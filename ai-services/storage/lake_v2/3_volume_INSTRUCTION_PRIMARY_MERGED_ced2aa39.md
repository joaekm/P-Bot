# Bestämning av konsultbehov och timvolym

För att bestämma antalet timmar per månad för två konsulter, behöver du först definiera om uppdraget är ett resursuppdrag eller ett projektuppdrag.

*   **Resursuppdrag:** Konsulterna arbetar under ledning av den avropande myndigheten. Timvolymen beror på myndighetens behov och kan variera.
*   **Projektuppdrag:** Leverantören ansvarar för att leverera ett resultat. Timvolymen beror på projektets omfattning och leverantörens planering.

Generellt sett kan man uppskatta timvolymen genom att:

1.  Definiera arbetsuppgifterna och deras omfattning.
2.  Uppskatta tidsåtgången för varje uppgift.
3.  Summera tidsåtgången för alla uppgifter per månad och konsult.

# Bestäm startdatum och resursbehov

För att fastställa när konsulten tidigast kan börja och hur länge resursen behövs, genomför följande steg i avropsprocessen:

1.  **Definiera behovet:** Identifiera tydligt behovet av IT-konsulttjänster, inklusive kompetensområde och konsultroll.
2.  **Specificera tidsperiod:** Ange önskat startdatum för konsultens uppdrag samt den beräknade längden på uppdraget.
3.  **Genomför avrop:** Följ avropsvägledningen för att genomföra ett avrop från ramavtalet, antingen via dynamisk rangordning eller förnyad konkurrensutsättning.
4.  **Kommunicera krav:** I avropsförfrågan, inkludera tydlig information om önskat startdatum och uppdragets varaktighet, så att leverantörerna kan lämna relevanta anbud.

# Uppskattning av totalt antal timmar för uppdrag

Dokumentet ger ingen specifik information om hur man uppskattar det totala antalet timmar för ett uppdrag. Det beskriver dock att uppdrag kan utföras som resursuppdrag eller projektuppdrag. Vid ett projektuppdrag har leverantören huvudansvaret för att leverera ett visst resultat i ett specificerat uppdrag. Vid ett resursuppdrag utför konsulten sitt arbete under ledning av den avropande myndigheten.

För att uppskatta det totala antalet timmar för ett uppdrag behöver man:

1.  Definiera uppdragets omfattning och mål.
2.  Identifiera de aktiviteter som krävs för att slutföra uppdraget.
3.  Uppskatta tidsåtgången för varje aktivitet.
4.  Summera tidsåtgången för alla aktiviteter för att få en total uppskattning.

**Exempel:**
Om varje konsult förväntas arbeta heltid, motsvarar det cirka 160 timmar per månad (40 timmar per vecka x 4 veckor). Två konsulter skulle då motsvara cirka 320 timmar per månad.

# Uppskattning av timmar för migreringsledare

För att uppskatta det totala antalet timmar som en migreringsledare kommer att behövas, utgå från om uppdraget är ett resurs- eller projektuppdrag.

*   **Resursuppdrag:** Konsulten arbetar under ledning av den avropande myndigheten. Uppskatta timmarna baserat på myndighetens projektplan och behov.
*   **Projektuppdrag:** Leverantören har huvudansvaret för att leverera ett visst resultat. Begär in en uppskattning av timmar från leverantören.

# Uppskatta total tid för projekt

För att uppskatta den totala tiden för ett projekt, beakta om uppdraget är ett resursuppdrag eller ett projektuppdrag. Vid **resursuppdrag** sker arbetet under ledning av den avropande myndigheten. Vid **projektuppdrag** har leverantören huvudansvaret för att leverera ett resultat. Tidsuppskattningen bör baseras på uppdragets omfattning och de resurser som krävs.

# Uppskattning av totala konsulttimmar

Ramavtalet specificerar inte en exakt metod för att uppskatta det totala antalet timmar en konsult kommer att arbeta under ett uppdrag. Avropande myndighet måste specificera omfattningen i sin avropsförfrågan. Leverantören lämnar sedan ett anbud baserat på denna förfrågan.

# Uppskattning av total tid för uppdrag

Dokumentet ger ingen direkt information om hur man uppskattar den totala tiden för ett uppdrag. Däremot beskrivs två typer av uppdrag:

*   **Resursuppdrag:** Konsulten arbetar under ledning av den avropande myndigheten.
*   **Projektuppdrag:** Leverantören har huvudansvaret för att leverera ett visst resultat.

För att uppskatta den totala tiden för ett uppdrag behöver man:

1.  Definiera uppdragets omfattning och mål.
2.  Identifiera de aktiviteter som behöver utföras.
3.  Uppskatta tidsåtgången för varje aktivitet.
4.  Summera tidsåtgången för alla aktiviteter.

Denna process bör göras i samråd med leverantören och den avropande myndigheten.

# Uppskatta totalt antal konsulttimmar

För att uppskatta det totala antalet timmar konsulten kommer att arbeta under projektets gång, behöver den avropande myndigheten specificera omfattningen av uppdraget i avropsförfrågan. Detta inkluderar att definiera om uppdraget är ett resursuppdrag eller ett projektuppdrag. 

Vidare bör man beakta:
*   **Resursuppdrag:** Konsulten arbetar under ledning av den avropande myndigheten.
*   **Projektuppdrag:** Leverantören har huvudansvaret för att leverera ett visst resultat.

Genom att specificera dessa faktorer kan en uppskattning av det totala antalet timmar göras.
