# Identifiera behov av IT-konsult

För att avgöra vilken typ av konsult som behövs, börja med att analysera behovet. Ramavtalet för IT-konsulttjänster 2021 omfattar sju kompetensområden: Verksamhetsutveckling och strategi, Ledning och styrning, Systemutveckling/-förvaltning och infrastruktur, Arkitektur, IT-, informations- och cybersäkerhet, Tillgänglighet och webb, samt Användarsupport/teknikstöd. Inom varje område finns exempelroller som IT-strateg, projektledare, systemutvecklare, arkitekt, säkerhetsexpert m.fl. Utgå från dessa för att precisera vilken kompetens som efterfrågas. Avrop kan ske genom dynamisk rangordning eller förnyad konkurrensutsättning.

Uppdrag kan utföras som resursuppdrag (konsulten arbetar under ledning av myndigheten) eller projektuppdrag (leverantören ansvarar för att leverera ett resultat).
