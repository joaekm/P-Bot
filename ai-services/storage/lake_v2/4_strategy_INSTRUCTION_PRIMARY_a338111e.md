# Bedömning av kvalitet vid utvärdering

För att bedöma kvaliteten på ett rättvist sätt med en viktning på 70% kvalitet och 30% pris, kan följande steg användas:

1.  **Definiera tydliga kvalitetskriterier:** Specificera vilka kvalitetsaspekter som är viktiga för uppdraget. Exempel kan inkludera erfarenhet, kompetens, referenser, kommunikationsförmåga och förståelse för uppdraget.
2.  **Använd en mervärdesmodell:** Tillämpa en utvärderingsmodell där prisavdrag erhålls för erbjuden kvalitet. Detta kan göras genom att definiera mervärden kopplade till de specificerade kvalitetskriterierna.
3.  **Genomför intervjuer:** Använd intervjuer för att bedöma konsultens kompetens och erfarenhet. Intervjufrågorna bör baseras på de krav och mervärden som ställts.
4.  **Referenstagning:** Kontrollera referenser för att verifiera konsultens tidigare erfarenhet och prestationer.
5.  **CV-bedömning:** Granska CV:n noggrant för att säkerställa att konsulten uppfyller de krav som ställts på kompetens och erfarenhet.
6.  **Objektiv utvärdering:** Säkerställ att utvärderingen genomförs objektivt och rättvist, med fokus på de definierade kvalitetskriterierna.
