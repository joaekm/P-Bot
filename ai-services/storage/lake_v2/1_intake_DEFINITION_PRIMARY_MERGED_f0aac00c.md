# Konsultroller inom kompetensområden

Ramavtalet för IT-konsulttjänster definierar ett antal kompetensområden med tillhörande exempelroller. För kompetensområde D (Stockholms län, Uppsala län och Gotlands län) kan du hitta information om relevanta roller i bilaga A, 'Kompetensområden och exempelroller'.

Exempel på roller som kan vara relevanta beroende på kompetensområde:

*   IT- eller Digitaliseringsstrateg
*   Modelleringsledare/Kravanalytiker
*   Metodstöd
*   Projektledare
*   Systemutvecklare/Systemintegratör
*   Enterprisearkitekt
*   Säkerhetsstrateg/Säkerhetsanalytiker
*   Webbstrateg
*   Teknikstöd – på plats
