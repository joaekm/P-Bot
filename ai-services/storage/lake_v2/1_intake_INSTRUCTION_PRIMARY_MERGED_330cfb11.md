# Verifiering av kunskap, erfarenhet och kompetensnivå samt beskrivning av uppdrag och arbetsuppgifter för konsult

Vid avrop med dynamisk rangordning kan bevis krävas för att verifiera kunskap om, erfarenhet av, och kompetensnivå för den aktuella rollen. Detta kan utgöras av:

*   Referenser eller annan dokumentation av liknande konsulttjänster gällande den/de konsult/er som ska utföra uppdraget
*   Intervju med den/de konsult/er som ska utföra uppdraget
*   CV för den/de konsult/er som ska utföra uppdraget
*   I förekommande fall säkerhetskontroll

För att få en detaljerad uppdragsbeskrivning och information om specifika arbetsuppgifter för en konsult, se följande avsnitt i avropsförfrågan:

*   **5.1 Uppdraget:** Här beskrivs uppdraget i detalj, inklusive vad konsulten ska göra, syftet med uppdraget och målsättningen.
*   **5.2 Roll:** Här beskrivs vilken roll/kompetens som efterfrågas.
*   **5.4 Krav på konsulten:** Här beskrivs kraven på resurskonsulten, inklusive kompetens, erfarenhet, referenser och CV.

# Ytterligare krav på referensuppdrag

Utöver de krav som redan anges i avropsförfrågan kan följande krav ställas på referensuppdraget:

*   **Verifiering av kunskap och erfarenhet:** Bevis som kan krävas för att verifiera kunskap om och erfarenhet av den aktuella rollen kan utgöras av:
    *   Referenser eller annan dokumentation av liknande konsulttjänster gällande den/de konsult/er som ska utföra uppdraget
    *   Intervju med den/de konsult/er som ska utföra uppdraget
    *   CV för den/de konsult/er som ska utföra uppdraget
    *   I förekommande fall säkerhetskontroll
*   **Kontakt med referenter:** För att kontrollera att angivna uppgifter är korrekta kan referenten komma att kontaktas via e-post eller telefon. Referenten ska vara vidtalad.

# Beskrivning av konsultens uppdrag

För att beskriva vad konsulten ska göra under uppdraget, ange följande i avropsförfrågan:

*   **Bakgrund till avropet:** Ange bakgrunden till varför konsulten behövs.
*   **Typ av uppdrag:** Om det är ett resurs- eller projektuppdrag.
*   **Ort för utförandet:** Var konsulten ska utföra uppdraget.
*   **Beskriv uppdraget:** Vad konsulten ska göra, syfte och målsättning med uppdraget.
*   **Roll:** Vilken exempelroll eller annan roll/kompetens som efterfrågas.
*   **Kompetensnivå:** Ange önskad kompetensnivå (1-5).
*   **Krav på konsulten:** Beskriv era krav på resurskonsulten, inklusive kompetens, erfarenhet, referenser och CV.

# Krav på certifieringar och språkkunskaper för konsulter

Avropande myndighet specificerar i avropsförfrågan vilka krav som gäller för avropet. Bevis som kan komma att krävas för att verifiera kunskap om och erfarenhet av den aktuella rollen kan komma att utgöras av:
* Referenser eller annan dokumentation av liknande konsulttjänster gällande den/de konsult/er som ska utföra uppdraget
* Intervju med den/de konsult/er som ska utföra uppdraget
* CV för den/de konsult/er som ska utföra uppdraget
* I förekommande fall säkerhetskontroll
