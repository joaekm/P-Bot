# Krav på certifiering, säkerhet och språk

Vid avrop kan specifika krav på certifieringar, säkerhet (exempelvis säkerhetskontroll) eller språkkunskaper behöva ställas på konsulten. Detta specificeras i avropsförfrågan. 

Vid förnyad konkurrensutsättning kan den upphandlande myndigheten ange vilka krav som gäller för avropet.
