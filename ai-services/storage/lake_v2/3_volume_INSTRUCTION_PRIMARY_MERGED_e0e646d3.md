# Flexibilitet i timuppskattning

Ramavtalet ger möjlighet att avropa resurser och projekt. Omfattningen av ett uppdrag kan variera beroende på behov, vilket påverkar den totala timåtgången. Vid avrop bör en noggrann kravspecifikation göras för att minimera risken för stora avvikelser från den initiala uppskattningen. Dialog med leverantörer kan ge en mer precis bild av tidsåtgången.
