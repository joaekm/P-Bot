# Intervjukriterier för konsult- och kandidatbedömning

Vid förnyad konkurrensutsättning kan intervjuer användas som ett kriterium för att bedöma konsulters kompetens och erfarenhet. Intervjufrågorna bör baseras på de krav och mervärden som ställts kopplat till kompetens och erfarenhet. Under intervjun kan följande utvärderas:

*   Kommunikativ förmåga
*   Förmåga att formulera sig väl
*   Förmåga att förmedla sina kunskaper på ett tydligt sätt.

Vid förnyad konkurrensutsättning kan följande kriterier för tilldelning av kontrakt komma att användas:

*   Pris
*   Kvalitet (t.ex. funktion som beskrivning av förväntat resultat eller användarvänlighet)
*   Kompetens och erfarenhet hos nyckelpersoner för uppdraget
*   Kundnöjdhet
*   Leveranstid för tjänster

Exempel på process:

1.  Genomför intervjuer med de konsulter som bedöms ha störst chans att tilldelas uppdraget baserat på offererat timpris och möjligt prisavdrag för erfarenhet.
2.  Intervjuerna kan genomföras på plats.
3.  Tidsram för intervjun:
    *   10 min – Konsulten får presentera sig
    *   50 min – Intervjufrågor
4.  Utvärdera mervärden under intervjun, exempelvis kommunikativ förmåga och förmåga att förmedla kunskaper.
