# Arbetsplats för migreringsledare

Avropande myndigheter är fria att utforma sina avrop hur de vill så länge dessa inte strider mot ramavtalets villkor eller mot lagen om offentlig upphandling (LOU). Detta innebär att det inte finns några specifika krav på att en migreringsledare måste sitta på plats, utan uppdraget kan utföras på distans eller hybrid om det passar både myndigheten och leverantören.
