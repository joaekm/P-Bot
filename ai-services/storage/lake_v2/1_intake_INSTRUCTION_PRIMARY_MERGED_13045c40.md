# Säkerställa konsulents närvaro på plats

För att säkerställa att konsulten är på plats 50% av tiden, specificera detta krav tydligt i avropsförfrågan. Ange i avtalet hur närvaron ska mätas och följas upp, exempelvis genom tidrapportering eller regelbundna avstämningar. Detta kan inkluderas som ett specifikt krav i utvärderingskriterierna vid förnyad konkurrensutsättning.
