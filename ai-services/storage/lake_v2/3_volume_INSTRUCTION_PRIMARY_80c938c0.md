# Fastställa startdatum för UX-specialist via avrop

För att fastställa när en UX-specialist tidigast kan börja, genomför ett avrop enligt Addas ramavtal för IT-konsulttjänster. Avropsförfrågan bör innehålla ett önskat startdatum eller en tidsperiod för när konsulten ska vara tillgänglig. Leverantörernas svar på avropsförfrågan kommer att indikera deras möjlighet att möta detta startdatum.
