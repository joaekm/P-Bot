# Specificera krav i avropsförfrågan

Vid avrop kan specifika krav på konsulten anges i avropsförfrågan, till exempel certifieringar eller språkkunskaper. Detta görs genom att:

1.  **Formulera egna roller:** Om de standardroller som finns inte täcker behovet, formulera egna roller som specificerar den kompetens som krävs.
2.  **Ange krav i avropsförfrågan:** Tydligt ange vilka krav som gäller för avropet, inklusive eventuella certifieringar eller språkkunskaper.
3.  **Definiera utvärderingskriterier:** Ange vilka utvärderingskriterier som kommer att ligga till grund för tilldelning av kontrakt, och hur dessa kommer att viktas.
