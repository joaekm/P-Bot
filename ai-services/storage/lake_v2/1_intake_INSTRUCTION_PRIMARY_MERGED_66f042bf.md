# Konsulttjänster kan utföras på plats eller på distans

Ramavtalet för IT-konsulttjänster tillåter att uppdrag utförs både som resursuppdrag och projektuppdrag. Konsulten/konsulterna kan utföra sitt arbete under ledning av den avropande myndigheten i den löpande verksamheten eller i ett specificerat projekt, vilket kan innebära arbete på plats. Alternativt kan konsulten/konsulterna utföra uppdraget på ett mer självständigt sätt, vilket möjliggör distansarbete.
