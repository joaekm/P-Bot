# Genomförande av uppdrag - distans, hybrid eller på plats

Uppdrag kan utföras som resursuppdrag eller projektuppdrag.

Ett **resursuppdrag** innebär att konsulten utför sitt arbete under ledning av den avropande myndigheten. Konsulten kan tillföra expertkompetens eller utgöra en resursförstärkning.

Ett **projektuppdrag** innebär att konsulten utför uppdraget mer självständigt och leverantören har huvudansvaret för att leverera ett visst resultat.

**Avropande myndigheter är fria att utforma sina avrop hur de vill så länge dessa inte strider mot ramavtalets villkor eller mot lagen om offentlig upphandling (LOU).** Detta innebär att det är möjligt att specificera krav på distansarbete, hybridarbete eller arbete på plats i avropsförfrågan.

# Val av avropsform

Avropande myndigheter är fria att utforma sina avrop hur de vill så länge dessa inte strider mot ramavtalets villkor eller mot lagen om offentlig upphandling (LOU). Ramavtalet ger möjlighet till både direktavrop (DR) och förnyad konkurrensutsättning (FKU).

# Utformning av avrop gällande arbetsplats

Avropande myndigheter är fria att utforma sina avrop hur de vill så länge dessa inte strider mot ramavtalets villkor eller mot lagen om offentlig upphandling (LOU). Detta innebär att det är möjligt att specificera om uppdraget ska utföras på plats, på distans eller som en hybridlösning.

Ramavtalet ger möjlighet att använda antingen direktavrop enligt dynamisk rangordning eller förnyad konkurrensutsättning.
