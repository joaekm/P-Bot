# Prissättning vid förlängning

Prissättningen för uppdrag, inklusive förlängningar, sker genom förnyad konkurrensutsättning (FKU). Detta ger möjlighet att anpassa prismodeller för det specifika uppdraget och skapa förutsättningar för förutsägbara projektkostnader.
